About SimpleQuiz

This project provide simple quiz throw questions.
It is developing for learning purposes.

Current features:

  1. Servlets.
  2. JSP.
  3. Html.
  4. Hardcoded questions.
  5. System.out.printLine logging.
  6. Session tracking.
  7. Usage of EL including custom tags.
  8. Scriplets free.

Planned improvements:

  1. Question storage in XML file on server.
  2. Time countdown (javascript?).
  3. Log4j.
  4. SVN.
  5. Admin page for editing available questions.
  6. Result storage in XML.
  7. ShowResults page
  8. JDBC.
  9. DB MySQL for question and results storage.
